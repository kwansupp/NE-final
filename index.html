Project

Modderkolk would like to become an ever-flourishing company that can double the revenue and quadruple the profit in a 4 year timespan. In order to reach this goal, we put our value in our people. This all begins by hiring people who are intrinsically motivated. To support them in doing their best work, we give them (i) a higher goal to strive for together, (ii) autonomy to make their own decisions, and (iii) development opportunities to further improve their skills and knowledge. Apart from this, to allow them to work more optimally, we continuously strive to improve our processes.

Vision

As this is a big project, it has been broken down to the following pieces below.
The initial project revolves around the first part and aims to create proof of concepts for the object recognition software.

Initial goals

From 12/12/2018

The goal set for July 2019 was:

  • - To have a working object classifier with over 95% confidence for a standard cabinet. This should be able to be used as a basis for a sorting robot.

  • - To have a system that has been tested with end-users, in this case, the logistics employees.

End results

As of 31/07/2019

What has now been achieved is:

  • - An object recognition system with an API (Application Programming Interface) that can be used to capture images, detect objects, and classify materials that are the main components of a standard cabinet (9 different objects).

  • - The developed system has not yet been user tested, but a value stream mapping has been completed with end-users in a meeting/workshop at the beginning phase of the project.

Overview of deliverables set in project charter.
The delivered hardware set up was for basic testing of software. For further development, more prototypes would have to be made; although this is out of the scope for this current project.
